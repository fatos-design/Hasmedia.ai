🔧 Meta App Setup Guide

Follow these steps to enable real Instagram API connections. Takes ~15 minutes. Once done, the "Connect Instagram" button in the dashboard will work for all clients.

1
Create a Meta Developer account
Go to developers.facebook.com and log in with your Facebook account. Click "Get Started" and complete the developer verification.
developers.facebook.com
2
Create a new App
Click "Create App" → Select "Business" as the app type → Name it "HasMedia Intelligence" → Click Create. You'll land on the app dashboard.
3
Add Instagram Graph API product
In your app dashboard, click "Add Product" → Find "Instagram Graph API" → Click Set Up. This gives you access to business account analytics.
4
Get your App ID and App Secret
Go to Settings → Basic. Copy your App ID and App Secret — you'll need these in the next step.
Settings → Basic → App ID + App Secret
5
Set your OAuth Redirect URI
In Facebook Login settings, add this as a Valid OAuth Redirect URI:
https://hasmedia.ai/auth/instagram/callback
6
Request these permissions
In App Review → Permissions, request the following scopes for your app:
instagram_basic · instagram_manage_insights · pages_read_engagement · pages_show_list
7
Send your App ID to the dev team
Once you have your App ID and Secret, send them securely to your developer. They'll wire it into the backend so the OAuth flow works live at hasmedia.ai.
Mark as complete when done
